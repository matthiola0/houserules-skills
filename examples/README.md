# Examples — filled-in `.ai-team/` configs

These are **complete, filled-in** reference configs to copy from when init asks you the
style questions. The blank templates with `<...>` placeholders live in
`skills/ceo/templates/`; here you can see what a finished version looks like.

| Example | Kind | What it shows |
|---------|------|---------------|
| [`typescript-web/`](typescript-web/.ai-team/) | code project | TypeScript + React stack: code style, comment style, doc/prose style, Conventional Commits |
| this repo's own [`.ai-team/`](../.ai-team/) | writing / docs project | Markdown skill pack: writing style, SKILL.md conventions, commit rules |

Copy the values you like into your own `.ai-team/` (or just answer init with them).
